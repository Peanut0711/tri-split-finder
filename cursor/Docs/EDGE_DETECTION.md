# 삼분할 엣지 검출 원리

`test/edge_check.py`에서 사용하는 엣지 검출 방식의 원리를 정리한 문서입니다.

---

## 1. 목적

**삼분할 영상**은 화면을 가로로 1/3, 2/3 지점에서 나눈 구도입니다.  
이 코드는 **1/3·2/3 위치에 수직 경계선(엣지)이 실제로 있는지** 수치로 측정해, 삼분할 여부를 판정합니다.

---

## 2. 전처리: 가우시안 블러

```python
blurred = cv2.GaussianBlur(frame_gray, (3, 3), 0)
sobel_x = cv2.Sobel(blurred, cv2.CV_64F, 1, 0, ksize=3)
sobel_x = np.abs(sobel_x)
```

- **역할**: 미세한 노이즈를 줄여서 Sobel이 **진짜 경계**에만 반응하도록 합니다.
- **방법**: (3×3) 가우시안 블러 적용 후, 그 결과에 Sobel을 적용합니다.

---

## 3. 엣지 강도: x방향 Sobel

- **Sobel (1, 0)**: **가로 방향 밝기 변화**를 측정하는 필터입니다.
  - **수직선**(왼쪽↔오른쪽 경계)에서 값이 크고,
  - 수평선이나 평평한 영역에서는 상대적으로 작습니다.
- **`np.abs(sobel_x)`**: 부호를 없애 “변화 크기”만 사용합니다.

→ 1/3·2/3 위치에 **수직선이 있으면** 그 부근에서 `sobel_x` 값이 커집니다.

---

## 4. “1/3·2/3 위치의 엣지 강도” 정의: get_peak_strength

```python
def get_peak_strength(center_x):
    lo = max(0, center_x - band_px)
    hi = min(w, center_x + band_px + 1)
    col_sums = np.sum(sobel_x[:, lo:hi], axis=0)  # 각 열(column)의 수직 합
    return np.max(col_sums)  # 그중 최댓값 = 피크
```

- **밴드**: `center_x ± band_px` 열(column)만 봅니다.
- **열별 수직 합**: `axis=0`으로 합치면 **각 열마다 “그 열 전체의 세로 합”**이 나옵니다.
- **반환값**: 그 열 합들 중 **최댓값** → “그 위치 근처에서 가장 강한 한 열의 수직 엣지 합”.

단순히 밴드 전체 합이 아니라 **피크(가장 선명한 수직선)**를 골라서, 삼분할 경계를 더 정확히 잡는 방식입니다.

---

## 5. 기준값: 로컬 참조 (배경 엣지 수준)

```python
ref_area_w = 20
# 왼쪽 1/3 선에서 왼쪽으로 100px 떨어진 20픽셀 폭
ref_l = np.sum(sobel_x[:, ref_l_lo:ref_l_hi]) / ref_area_w
# 오른쪽 2/3 선에서 오른쪽으로 100px 떨어진 20픽셀 폭
ref_r = np.sum(sobel_x[:, ref_r_lo:ref_r_hi]) / ref_area_w
local_ref = max(ref_l, ref_r) * threshold_mult
```

- **참조 영역**: 삼분할 경계선이 아닌 **일반 배경/내용**이 있는 구간(경계에서 100px 떨어진 20픽셀 폭).
- **단위 맞춤**: 20픽셀 폭의 합을 20으로 나눠 **“열 하나당 평균 수직 합”**으로 만들고, `get_peak_strength`가 반환하는 단위(한 열의 수직 합)와 맞춥니다.
- **`local_ref`**: 위 참조값의 `threshold_mult`배 → “배경보다 이만큼 강해야 경계선으로 인정”하는 기준입니다.

---

## 6. 판정

- **pass_left**: 1/3 위치 피크 강도 > `local_ref`
- **pass_right**: 2/3 위치 피크 강도 > `local_ref`
- **삼분할**: `pass_left and pass_right` → 두 수직선 모두 기준을 넘으면 삼분할 영상으로 판정합니다.

---

## 요약

| 단계 | 내용 |
|------|------|
| 1 | 가우시안 블러로 노이즈 감소 후 **x방향 Sobel**로 수직 엣지 강도 계산 |
| 2 | **1/3, 2/3** 위치에서 `band_px` 밴드 내 **열별 수직 합의 최댓값**으로 해당 위치의 “엣지 강도” 정의 |
| 3 | 경계선에서 100px 떨어진 **배경 구간**의 열당 평균 sobel 합으로 **로컬 참조** 계산 (단위 맞춤) |
| 4 | 참조 × `threshold_mult`를 넘으면 “수직 경계 있음”으로 보고, **양쪽 모두 넘을 때만** 삼분할로 판정 |

정리하면, **“1/3·2/3에 배경보다 충분히 강한 수직 엣지가 있는가?”**를 수치로 검사하는 방식입니다.
